FAQ:

Где взять ODBC Driver?

Текущая версия требует для подключения к базе данных ODBC драйвер. Необходима 32-битная версия ODBC драйвера.

Для SQLite драйвер можно скачать тут.

Для PostgreSQL рекомендуется установить ODBC драйвер через идущий в комплекте с PostgreSQL Application Stack Builder.

Почему ORM это плохо?

ORM не позволяет точно контролировать извлечение данные и обрабатывать сложные документы у которых нет чёткой схемы.

Так же большинство ORM не позволяет сохранять пространственные отношения родитель-потомок между узлами.

Нужны ли XSD схемы?

Практика показывает, что XSD схемы создают больше проблем чем решают. Далеко не все XML файлы имеют XSD схемы. Далеко не все схемы корректны.

Что такое грамматики и как писать правила NLP?

В SmartXML используется встроенный в язык Red движок Parse который позволяет выполнять ряд достаточно сложных манипуляций с данными. Движок крайне легковесный, поэтому он не может идти в сравнение с полновесными NLP движками, однако он крайне быстрый и позволяет делать то, что невозможно сделать с помощью тех же регулярных выражений.

Вы можете написать правила самостоятельно изучив документацию Parse или заказать написание правил у нас.
